# 🤖 AI Agent Tool - 智能助手

## 產品描述
一個基於 AI 嘅自動化工具，可以幫你完成重複性任務。

## 功能

### 1. Email 自動回覆
- 智能分類 incoming emails
- 根據上下文生成回覆
- 自動標記 action items

### 2. 內容生成
- 博客文章
- 社交媒體帖子  
- 產品描述

### 3. 數據分析
- 自動生成報告
- 趨勢分析
- 圖表生成

### 4. 會議助手
- 自動生成會議紀錄
- 提取 action items
- 發送 follow-up emails

## 技術棧
- Backend: Node.js + Express
- AI: MiniMax API
- Database: SQLite (簡單部署)
- Frontend: React
